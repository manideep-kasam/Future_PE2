# AI Video Ad Generation Prompt

## Overview

This project provides a comprehensive AI prompt for generating high-quality, cinematic video advertisements. It enables AI models to create detailed, scene-by-scene video concepts complete with camera directions, voice-over scripts, on-screen text, audio cues, and marketing frameworks. The output is optimized for short-form social media platforms and ready for AI video generation tools.

## Features

* Generates complete video ad concepts
* Creates scene-by-scene production prompts
* Includes cinematic camera angles and movements
* Produces voice-over scripts and on-screen text
* Adds background music and sound effect suggestions
* Follows proven ad frameworks (Hook → Problem → Solution → Benefits → CTA)
* Optimized for vertical 9:16 short-form videos
* Creates thumbnail ideas, captions, and hashtags
* Compatible with popular AI video generation platforms

## Input Requirements

Provide the following information:

* Product or Brand Name
* Product Category
* Target Audience
* Unique Selling Proposition (USP)
* Key Features
* Main Benefits
* Customer Pain Points
* Desired Customer Transformation
* Brand Personality
* Brand Colors
* Platform (Instagram Reels, TikTok, YouTube Shorts, Facebook Ads)
* Preferred Video Duration
* Call-to-Action

If some information is unavailable, the prompt makes realistic assumptions and clearly identifies them.

## Output

The prompt generates:

* Video Concept
* Scene-by-Scene Breakdown
* Camera Angles and Movements
* Lighting and Visual Style
* Voice-Over Script
* On-Screen Text
* Audio and Music Suggestions
* Marketing Framework-Based Flow
* AI Video Generation Prompt
* Thumbnail Idea
* Social Media Caption
* Relevant Hashtags
